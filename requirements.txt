# mtb-map-framework Python dependencies — pinned for reproducible builds.
#
# Requires Python 3.11+ (tested through 3.14). The only direct deps are those
# under "Direct dependencies" below; everything else is a pinned transitive (captured via
# `pip freeze`) so a fresh `pip install -r requirements.txt` six months
# from now produces the same dependency graph and the same build output
# regardless of upstream releases.
#
# To upgrade:
#   .venv/bin/pip install -U <pkg> && .venv/bin/pip freeze > requirements.txt
# Then build all maps and confirm no regressions before committing the
# updated lockfile.
#
# OS-level dependencies (NOT installed by pip):
#   - pmtiles CLI (Go binary) — for basemap/terrain PMTiles extraction.
#     Install via Homebrew: `brew install pmtiles` (macOS), or build from
#     https://github.com/protomaps/go-pmtiles releases.
#   (Pillow ships manylinux/macOS wheels that bundle libjpeg/zlib, so no
#    system image libraries need to be installed for `pip install`.)
